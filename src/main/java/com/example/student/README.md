# Student Management System (CRUD)

A simple **Student Management System** built using **Spring Boot**, **Spring Data JPA**, **MySQL**, and **HTML + JavaScript**.  
This project supports full **CRUD operations** (Create, Read, Update, Delete) for student records.

---

## Features

- Add new students
- View all students
- Search student by ID
- Update student details
- Delete student records
- REST API + Simple Frontend
- Data stored in MySQL database

---

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA (Hibernate)
- MySQL
- HTML
- JavaScript
- Maven
- Git & GitHub
